# VoltX — Gen Z Electronics Brand

This is a front-end prototype for a Next-Gen Electronics brand, built with **Next.js 15**, **Tailwind CSS**, and **Zustand**.

## Features
- 🌐 **Fully Responsive**: Mobile-first design for M-web and Desktop.
- 🕹️ **Cyberpunk Aesthetic**: Dark mode, glassmorphism, and neon accents.
- 🛒 **Functional Cart**: Slide-out cart with real-time total updates.
- 💳 **Razorpay Flow**: Interactive payment gateway simulation.
- 📦 **Product Pages**: Dynamic routing for electronic products.

## Getting Started

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
